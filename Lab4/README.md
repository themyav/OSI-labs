## Lab 4: инструкция по запуску

### Сервер:

```g++ -o server server.c```

### Модуль

```chmod u+x ./start```

```chmod u+x ./exit```

```./start```

Чтобы остановить: 

```./exit```

Просматривать логи ядра в режиме реального времени:

```sudo dmesg -w```

С высокой вероятностью оно положит вашу систему после выполнения 3-5 команд((
